Resumen

El proyecto YC se desarrolló en el lenguaje java, junto con el gestor de base de datos access para dar una mayor familiaridad al cliente, junto a otras técnologias de igual importacia como los sockets.

Java

Java es uno de los lenguaje más conocidos en la actualidad esto es innegable, se decidio realizar el proyecto con java por su versatilidad y su robustosidad.

Access

Su interfaz amigable hace a access el gestor de base de datos uno de los más sencillos de utilizar, por eso se decanto por este para el proyecto asi como que este gestor no es tan dependiente de SQL para operar

Otras tecnologias

Sockets

Los socket son la parte más fundamental del proyecto, ya que por este modo se conectaron las dos aplicaciones realizadas
